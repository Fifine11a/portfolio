SwapApp

Visit SwapApp on Netlify

SwapApp is my exit project of the Czechitas Digital academy. We worked as a couple with my partner Magdalena. It was our first independent project. We had mentors advising us, but we were responsible E2E. We created the first web app for swapping in Czech Republic. The goal was a minimalistic, simple, user friendly, mobile first app.

how we did it?

Pure CSS, React, React Router, FireBase

where we are?

After fixing bugs, discovered by testing, we move to the second development phase. The main goal will be development of a search tool, edit/ delete option and communication between users with automatically generated emails.
